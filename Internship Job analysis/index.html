Project Overview

This project analyzes internship and job posting data using Tableau. The dashboard includes a Training Dashboard, Internship Dashboard, and six analytical tasks covering hiring trends, salary distribution, company analysis, regional job comparisons, and geographic job mapping.


The project was developed using Tableau Public and follows all assignment requirements without modifying the original dataset. All visualizations were created using Tableau dashboards and stories, with detailed filter documentation and analytical insights.
