# Internship Job Analysis Project

## Project Overview
This project analyzes internship and job posting data using Python, Tableau Public, HTML, and Netlify. The objective is to explore internship trends, company hiring patterns, salary distributions, qualification requirements, and geographical comparisons through interactive dashboards.

## Project Workflow
1. Data Cleaning and Preprocessing using Python
2. Data Analysis and Transformation
3. Dashboard Development using Tableau Public
4. Story Creation with Multiple Analysis Tasks
5. Website Integration using HTML
6. Deployment using Netlify

## Technologies Used
- Python
- Pandas
- NumPy
- Tableau Public
- HTML
- Netlify

## Project Components

### Python
- Data cleaning and preprocessing
- Handling missing values
- Data transformation and filtering

### Tableau Dashboards
- Training Dashboard
- Task 1: Preference vs Work Type
- Task 2: Company Size vs Company Name
- Task 3: Internship Salary Distribution
- Task 4: India vs Germany Job Comparison
- Task 5: Top 10 Companies Hiring Comparison
- Task 6: Qualification Drilldown Analysis

### Web Deployment
The Tableau dashboard is embedded into a custom HTML webpage and deployed using Netlify.
